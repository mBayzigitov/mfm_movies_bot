# Movies Bot
## Made by mBayzigitov

MoviesForMe is a bot with which you can

- Find out premieres in the box office
- Learn more about the movie
- Find information about an actor or director
- Find a movie similar to the one you liked

## Commands

- /start
- /premieres
- /filminfo
- /person
- /similars
